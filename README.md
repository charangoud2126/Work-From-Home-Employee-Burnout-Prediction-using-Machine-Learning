# 🧠 Work From Home Employee Burnout Prediction

### Predicting Employee Burnout using Machine Learning

## Problem Statement

With the rise of remote work, employee burnout has become a silent productivity killer. Organizations struggle to identify burnout early and take preventive action.

This project builds a Machine Learning model to predict employee burnout risk, enabling companies to make data-driven decisions.

## Objective

Predict employee burnout risk accurately

Identify key factors contributing to burnout

Provide actionable insights for HR & management

## Key Results

* 95% Accuracy using Random Forest & SVM

* Strong correlation between mental fatigue & burnout

* Clear impact of work hours and resource allocation

## Tech Stack

Model	Accuracy

## Key Insights

* Mental Fatigue Score → Strongest predictor

* Long working hours → Higher burnout risk

* Balanced resource allocation → Lower burnout

* Remote work requires structured work-life balance
